## 安装环境
直接参考laravel的环境部署方案，运行在虚拟机上
https://learnku.com/docs/laravel-development-environment/6.x/development-environment-windows/5518

1. 安装VirtualBox
2. 安装Vagrant
3. 导入Homestead Box 虚拟机盒子
4. 安装Git
5. 安装Homestead 管理脚本
6. 配置 ``Homestead.yaml``  文件
7. 启动 Homestead 虚拟机
8. 开启 NFS 挂载模式
